# manito

## install
- firebase 설치 필요.
  `npm i firebase`
- 최상위에 .env파일 생성 후 firebase관련 키 입력.
- vue-router 설치 필요.
  `npm i vue-router`
- vuex 설치 필요.
  `npm i vuex --save`
- vuex-persistedstate 설치 필요.
  `npm i vuex-persistedstate`
